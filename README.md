# Examen-2-POO
Proyecto Examen 2 de la materia de Programacion Orientada a Objetos
Integrantes: Marcos Araya, Said Quinto

Pasos para ejecutar el servicio:
1-. Descargue el proyecto del repositorio de github
2-. Hostee el proyecto en su IDE
3-. Una vez hosteado con exito ingrese a la pagina de servicio
            https://la-tiendita-api.netlify.app/
4-. Realice las acciones que mas guste.

puede consumir las apis del proyecto integrandolo con otro tipo de frontend
tambien puede cambiar el tipo de base de datos realizando la conexion en 
application.properties, la guia para las solicitudes de las apis se encuentran 
tambien en el documento endpoits.
